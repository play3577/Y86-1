### Y86 CPU模拟器：

![Y86](/home/aaron/Desktop/Y86.png)

###功能实现：

#### 1.汇编器

* 读取.ys文件
* 生成.yo文件

#### 2.Y86内核

* Y86基本指令集
* CMOVXX指令
* 数据转发
* 分支预测错误
* 处理冒险

#### 3.Y86界面

* 时钟控制
* 程序员可见状态
  * Reg / Mem / CC / PC / Stat
  * 内存地址搜索（可转换进制）
* 流水线寄存器
  * 变化数据高亮
* 代码框
  * .yo代码（直接读取，或由.ys转换）
* 流水线模拟
  * 纵行 - 周期
  * 横向 - 指令
  * 浮窗 - 操作细节
